# Quick Start

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Play**:
   - Open [http://localhost:3000](http://localhost:3000).
   - As the **Human** player, click "Roll Dice" to start your turn.
   - Watch the AI players take their turns automatically!
   - View their banter in the Chat Log.

4. **Controls**:
   - **Roll Dice**: Start your turn.
   - **Buy**: Purchase an unowned property you landed on.
   - **Skip**: Pass on the property.
   - **End Turn**: Finish your turn.
